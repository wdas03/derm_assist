# DermAssist+

Take a photo, upload it, get a diagnosis, and connect with recommended dermatologists.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
